Суть: Создать одностраничное веб-приложение (SPA на чистом JS или Vue/React), которое имитирует интерфейс управления задачами. Бэкэнд писать не нужно — используйте LocalStorage браузера для имитации базы данных.

Логика работы:
Интерфейс: Слева — колонка для добавления новой задачи (Название, Стек, Описание). Справа — список уже созданных задач в виде карточек.

Функционал:
При нажатии на кнопку «Создать задачу» она должна мгновенно появляться в списке без перезагрузки страницы.
Реализовать фильтрацию: сверху кнопочки-теги (например: «Python», «Web», «Дизайн»). При клике на тег в списке остаются только соответствующие задачи.
Реализовать удаление карточки задачи.

Технические требования:
Верстка: Чистый и адаптивный HTML/CSS (карточки должны аккуратно перестраиваться в одну колонку на мобилках).
JS: Работа с массивами объектов, событиями (Click, Submit) и DOM-деревом.
Стиль: свободный

Бонусное задание:
Реализовать поднятие проекта через Docker и Docker compose

Срок: 1-2 дня. Как истекает срок, вы должны предоставить ссылку на GitHub с кодом или архив с файлами. Мы оценим любой ваш результат. Задание призвано не ужесточить отбор кандидатов, а точнее понять ваши скиллы, чтобы подобрать для вас подходящие проекты в будущем.
